Technical

Linux

I am used to configuring and modifying Linux to install software, run startup tasks, build code, and deploy applications.

An example of this is a ‘Vagrantfile’ including setup shell scripts to build and deploy an Apache Tomcat application that I wrote. ZIP files of this project can be provided upon request as this code resides in a private repository.
